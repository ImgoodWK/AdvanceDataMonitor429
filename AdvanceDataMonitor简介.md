# AdvanceDataMonitor 模组简介

**AdvanceDataMonitor** 是一个专为 **GTNH（GregTech: New Horizons）** 整合包设计的 Minecraft 1.7.10 模组，为 **Applied Energistics 2（AE2）** 网络提供强大的数据监控、可视化和 AI 辅助功能。

## 核心功能

### 📊 数据监控与可视化
- **高级数据监视器**：在游戏世界中实时显示 AE2 网络数据，支持折线图、柱状图、3D 柱状图、瀑布流、势差图等多种图表类型
- **数据织取器**：绑定任意方块并查看其 NBT 数据，帮助发现可监控的字段
- **灵活的显示配置**：支持自定义标题、颜色、透明度、位置、旋转和缩放

### 🔗 AE2 网络集成
- **AE 网络链接**：统计整个 AE2 网络的物品/流体存储容量和使用情况
- **AE 合成链接**：监控合成 CPU 数量、忙碌状态、存储和协处理器信息
- **AE 存储链接**：按配置的"高级存储链接元件"显示指定物品的库存数量，支持模糊卡、反相卡、矿典卡和流体标记

### 🤖 AI 智能助手
- **自然语言查询**：用中文或英文询问 AE2 存储情况，如"查一下 AE 里有多少锡锭"
- **智能合成下单**：支持批量合成订单，如"帮我合成 64 个锡齿轮，32 个铜线"
- **物品取出**：直接从 AE2 存储取出物品到背包，如"取出 64 个锡锭到背包"
- **多供应商支持**：兼容 DeepSeek、OpenAI、OpenRouter 等多种 AI 服务提供商

### 🎤 语音助手
- **语音输入**：通过快捷键录音，支持本地 Vosk 模型或云端 Whisper 服务
- **无缝集成**：语音转文字后自动进入 AI 助手流程

## 技术特点

- **Minecraft 版本**：1.7.10
- **Forge 版本**：10.13.4.1614
- **模组 ID**：advancedatamonitor
- **主要依赖**：GT5-Unofficial、AE2FluidCraft-Rework、NewHorizonsCoreMod 等

## 主要方块与物品

| 名称 | 功能 |
|------|------|
| 高级数据监视器 | 主显示方块，支持多种数据可视化 |
| 数据织取器 | 绑定方块查看 NBT 数据 |
| 高级网络链接 | 接入 AE2 网络统计存储状态 |
| 高级合成链接 | 监控 AE2 合成 CPU 状态 |
| 高级存储链接 | 按配置显示指定物品数量 |
| 高级存储链接元件 | 存储链接的过滤配置单元 |

## 使用场景

1. **AE2 网络监控**：实时查看存储使用趋势，制作网络状态看板
2. **合成进度跟踪**：监控多个合成 CPU 的工作状态
3. **关键材料库存墙**：集中显示重要物品的库存数量
4. **AI 辅助操作**：用自然语言查询存储、提交合成订单、取出物品
5. **语音控制**：通过语音命令快速执行 AE2 操作

## 安装与配置

1. 确保安装 Minecraft Forge 10.13.4.1614
2. 安装所有依赖模组
3. 将 AdvanceDataMonitor.jar 放入 mods 文件夹
4. 启动游戏生成配置文件
5. 如需 AI 功能，在配置文件或游戏内设置 API Key
