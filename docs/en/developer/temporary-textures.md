> Audience: developers / artists · Last updated: 2026-06-24

This document lists **missing or placeholder** block/item textures in **TeXTech**. Entries marked **temporary** were generated by `.workspace/generate_temp_textures.py` using the mod’s existing pixel-art language (dark blue base, cyan highlights, red/gold accents, 64×64 cell frames). They are **for development and placeholder use only** and should be replaced with hand-authored art before release.
