## 重新生成

```bash
python .workspace/generate_temp_textures.py
```

生成后需将方块用图标复制到 `textures/blocks/`（脚本不会自动复制，见上文表格）。

## 后续正式美术待办

1. **数据映录器**（`data_weave`）：与 `DataWeave.obj` 模型 UV 统一的工具造型；当前缺 `model/DataWeave.obj`，物品 3D 渲染仍会 fallback。
2. **高级存储链接元件**（`advance_storage_link_cell`）：对齐 AE2 存储元件视觉语言，区分于织形/织尘元件。
3. **织潮元件**（`data_tide_loom_cell`）：在涌流元件基础上增加独立「潮汐」识别元素，而非仅改色。
4. **挂索器 / 挂索节点**：挂钩、锚点、缆线语义需更清晰。
5. **三种 AE2 链接器方块**：物品栏图标可与 TESR 造型统一；Network Link 已有 `AdvanceNetworkLink.obj`，可考虑专用 UV 图集而非共用 `TeXTech.png` 模型贴图。

## 维护约定

- 替换正式材质时：**删除或覆盖**对应 PNG，并从本文 **「本次补齐的临时材质」** 表中移除该行。
- 新增方块/物品注册后：在 PR 中检查 `textures/items/` 与 `textures/blocks/` 是否存在对应文件，并更新本文与英文版。
