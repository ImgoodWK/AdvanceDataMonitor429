## 1. 物品简介

**高级计划器**（Advance Planner）是 AdvanceDataMonitor 模组提供的一个轻量级任务规划工具。你可以把它理解为一个"游戏内的待办清单"——在其中记录你需要完成的事项，逐条勾选，并支持将多个计划器的内容整合到一起。

**核心特性：**

- 每个计划器可存储**多条待办条目**
- 每条条目支持**文本描述**和**完成状态**
- 支持**滚动翻页**，最多可容纳 50 行显示
- 支持将背包中所有计划器**整合为一个**
- 支持在屏幕 HUD 上显示待办列表

## 3. 基本操作

### 3.1 打开计划器 GUI

**右键点击**（不潜行）手持的高级计划器，即可打开计划器界面。

### 3.2 添加条目

1. 打开 GUI 后，你会看到一个列表区域，最多显示 8 行（可通过滚轮翻页查看更多）。
2. **点击一个空行**（显示"点击添加..."的行），会弹出文本输入框。
3. 输入你想要记录的内容，然后点击绿色的 **"添加"** 按钮确认，或按 **Enter** 键确认。
4. 如果不想添加，点击红色的 **"X"** 按钮取消，或按 **Esc** 键取消。

> **注意：** 如果输入框为空就确认，新条目不会被创建；但清空已有条目的内容后确认，会保留该条目（文本变为空）。

### 3.3 编辑条目

**点击已有条目的文本区域**，即可进入编辑模式。编辑完成后按 Enter 或点击"添加"按钮保存。

### 3.4 勾选完成 / 取消完成

- 每行最左侧有一个 **复选框**。
- **点击复选框** 即可切换该条目的完成状态。
- 已完成的条目会显示为**绿色**并带有**删除线**效果。
- 未完成的条目显示为**白色**。

> 鼠标悬停在复选框上时，会显示提示文字"标记为已完成"或"标记为未完成"。

### 3.5 滚轮翻页

当条目数量超过一屏（8 行）时，使用**鼠标滚轮**上下滚动列表。右侧会显示滚动条指示当前位置。

## 4. GUI 界面详解

### 4.1 主界面布局

```
┌─────────────────────────────────────────┐
│           §l高级计划器                    │  ← 标题
│  共 N 条 | 完成 X | 未完成 Y              │  ← 统计信息
├─────────────────────────────────────────┤
│ ☐ #1  采集64个铁锭                14:30  │
│ ☑ #2  建造仓库                     14:31  │  ← 已完成（绿色+删除线）
│ ☐ #3  制作钻石镐                   14:32  │
│ ☐ #4  点击添加...                         │  ← 空槽位
│ ...                                      │
├─────────────────────────────────────────┤
│     [ 整合 ]              [ 退出 ]       │  ← 底部按钮
└─────────────────────────────────────────┘
```

### 4.2 各元素说明

| 元素 | 说明 |
|------|------|
| **标题** | 固定显示"高级计划器" |
| **统计信息** | 显示总条目数、已完成数、未完成数 |
| **序号 (#N)** | 每条条目的唯一编号，从 1 开始递增 |
| **复选框** | 点击切换完成/未完成状态 |
| **文本区域** | 条目的文字内容，点击可编辑 |
| **时间戳** | 每条右侧显示添加时间（格式：`yyyy-MM-dd HH:mm`） |
| **整合按钮** | 跳转到整合确认界面（详见[第5节](#5-整合功能合并多个计划器)） |
| **退出按钮** | 关闭 GUI 并返回游戏 |

### 4.3 按钮列表

| 按钮 | 颜色 | 功能 |
|------|------|------|
| 整合 (Merge) | 青色 | 进入计划器整合界面 |
| 退出 (Exit) | 青色 | 关闭界面 |
| 添加 (Add) | 绿色 | 确认当前编辑的文本 |
| X | 红色 | 取消当前编辑 |

## 5. 整合功能（合并多个计划器）

当你背包中有**多个**高级计划器时，可以将它们的内容合并到一个计划器中。

### 5.1 操作步骤

1. 确保背包中有 **2 个或以上** 高级计划器。
2. 右键打开其中任意一个计划器。
3. 点击底部的 **"整合"** 按钮。
4. 进入整合确认界面，会显示：
   - 背包中计划器的数量和总条目数
   - 每个计划器的条目数量（当前打开的会标注"（当前）"）
5. 选择合并模式（见下文）。
6. 点击 **"确认整合"** 执行合并。

### 5.2 合并模式

| 模式 | 说明 |
|------|------|
| **按时间合并** (Merge by Time) | 将所有条目按添加时间排序，时间早的排在前面 |
| **按序号合并** (Merge by Index) | 保持原计划器的条目顺序，先放第一个计划器的全部条目，再追加第二个的，依此类推 |

### 5.3 合并规则

- 合并后，所有条目会重新编号（从 1 开始连续编号）。
- 每条条目的**文本内容**、**完成状态**和**原始时间戳**都会被保留。
- 合并操作会**消耗**其他计划器物品（从背包中移除），只保留当前这个。
- 合并完成后自动回到计划器主界面，可立即查看结果。

### 5.4 取消整合

在整合确认界面点击 **"取消"** 按钮，将返回计划器主界面，不会执行任何合并操作。

## 6. HUD 显示开关

高级计划器支持在游戏屏幕上以 HUD（抬头显示）的形式展示待办列表。

**切换 HUD 显示：**

- **潜行 + 右键** 手持的高级计划器，即可开关 HUD 显示。
- 开启时聊天栏会显示"HUD 已启用"（青色文字）。
- 关闭时聊天栏会显示"HUD 已关闭"。

## 8. 使用技巧

1. **分类管理：** 准备多个计划器，分别用于不同目的（如"建造计划"、"材料清单"、"探索目标"），需要时再整合。

2. **利用完成状态：** 完成的条目带有删除线效果且颜色不同，方便快速区分进度。

3. **整合前检查：** 整合前可以在确认界面查看每个计划器的条目数，确认无误再操作。

4. **HUD 实用场景：** 在建造大型项目时，潜行+右键开启 HUD，就能在游戏画面上随时看到待办列表，无需反复打开 GUI。

5. **不可堆叠：** 每个高级计划器占用一个物品栏位（最大堆叠数为 1），请注意背包空间。

## 9. 常见问题

**Q: 整合按钮灰色/不可用？**

A: 确保背包中至少有 2 个高级计划器。只有 1 个时无法整合。

**Q: 合并后原来的计划器去哪了？**

A: 合并会消耗掉其他计划器，只保留当前打开的那个。合并前请确认不需要保留原始物品。

**Q: 条目数量有上限吗？**

A: 显示区域最多同时展示 50 行（通过滚动翻页），实际存储的条目数量理论上没有硬性限制（受 NBT 数据大小约束）。

**Q: 编辑时按 Esc 会怎样？**

A: 按 Esc 会取消当前编辑，不会保存你正在输入的内容。

**Q: 关闭 GUI 时未保存的编辑会丢失吗？**

A: 不会。关闭 GUI 时，如果有正在编辑的条目，系统会自动提交当前编辑内容。
